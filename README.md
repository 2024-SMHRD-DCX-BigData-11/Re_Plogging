## 📢 서비스 소개
* 서비스명: 플로깅 및 쓰레기 배출 감소를 위한 웹 서비스<br>
* 서비스설명: 플로깅 활동을 하고 싶은 사람들과 쓰레기 분리배출에 대해 궁금증을 풀고 싶은 사람들을 위한 친환경 종합 플랫폼 서비스
<br><br><br>

## 📅 프로젝트 기간
